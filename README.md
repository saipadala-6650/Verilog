# 🚀 60 Days of Verilog Coding Challenge

## 📌 Overview

This repository documents my **60-day coding journey in Verilog HDL**, where I practiced implementing digital logic concepts through RTL design and simulation.

The goal of this challenge was to strengthen my understanding of **digital electronics concepts and their hardware implementation using Verilog**, along with analyzing **simulation waveforms** using EDA tools.

During these 60 days, I implemented multiple **combinational circuits, sequential circuits, and Finite State Machines (FSMs)**.

---

## 🎯 Objectives

* Build strong fundamentals in **Verilog HDL**
* Implement **digital circuits using RTL modeling**
* Understand **simulation and waveform verification**
* Develop consistency through **daily coding practice**

---

## 🧠 Topics Covered

### 🔹 Verilog Fundamentals

* Module structure
* Data types (`wire`, `reg`)
* Continuous assignments
* Procedural blocks (`always`, `initial`)
* Blocking vs Non-blocking assignments
* Conditional statements (`if`, `case`)

### 🔹 Combinational Circuits

Implemented and simulated several combinational logic designs:

* Multiplexer
* Demultiplexer
* Encoder
* Decoder
* Comparator
* Adders and Subtractors
* Code Converters

### 🔹 Sequential Circuits

Sequential logic implementations including:

* SR Flip-Flop
* JK Flip-Flop
* D Flip-Flop
* T Flip-Flop
* Counters
* Shift Registers
* Ring Counters

### 🔹 Finite State Machines (FSM)

Design and simulation of FSM-based systems:

* Moore FSM
* Mealy FSM
* Traffic Light Controller
* Control-based sequential designs
